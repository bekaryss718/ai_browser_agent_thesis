============================================================
  BROWSER AGENT - QUICK START GUIDE
============================================================

FIRST TIME SETUP (do this once):
----------------------------------
1. Double-click  INSTALL.bat
   Wait 3-5 minutes. Internet required.

2. Open the file  .env  with Notepad
   Replace:  sk-ant-YOUR_KEY_HERE
   With your real key from: https://console.anthropic.com
   Save the file.

3. Double-click  START.bat
   Browser opens automatically.


EVERY DAY:
-----------
Double-click  START.bat
Browser opens automatically at http://localhost:8000


ACCOUNTS:
----------
  admin / admin123  - full access (all users and tasks)
  alice / alice123  - regular user
  bob   / bob123    - regular user


STOPPING THE APP:
------------------
Press  Ctrl+C  in the black window, then close it.


TROUBLESHOOTING:
-----------------
  Not installed error  -> run INSTALL.bat again
  API key error        -> check your .env file
  Page won't open      -> make sure START.bat is running
  Agent fails          -> run INSTALL.bat again
